# SISKIN — Madárhegy Residence

Interaktív, reszponzív 3D ingatlanbemutató koncepció egy fiktív budapesti társasházhoz.

## Funkciók

- forgatható és nagyítható procedurális 3D épületmodell
- nappali/esti nézet és interaktív fénytanulmány
- négy lakás szűrhető listával és részletes alaprajzzal
- kedvencek és ajánlatkérő folyamat
- helyben futó, természetes nyelvű AI otthonajánló
- mobil- és asztali felület

## Fejlesztés

```bash
npm install
npm run dev
```

Production build:

```bash
npm run build
```

## Technológia

React, TypeScript, Vite, Three.js, React Three Fiber és Framer Motion.

> A projekt és minden benne szereplő ingatlanadat fiktív koncepció, nem minősül ajánlattételnek.
